# TextUtils - Django Web App

TextUtils is a simple text utility web application built using Django.
This website allows users to perform different operations on their text easily.

## Features

* Convert text to **UPPERCASE**
* **Remove new lines**
* **Remove extra spaces**
* **Count characters**
* **Remove punctuations**

## Technologies Used

* Python
* Django
* HTML
* CSS

## Installation

Follow these steps to run the project locally:

1. Clone the repository

```
git clone https://github.com/munazirabbas47-dotcom/textutils-django.git
```

2. Go to the project folder

```
cd textutils
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the server

```
python manage.py runserver
```

5. Open in browser

```
http://127.0.0.1:8000/
```
